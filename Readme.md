# 📊 Student Record Management System

A simple command-line student record management system made using Python and Pandas. I built this project mainly to practice working with CSV files, DataFrames, functions, file handling, and basic data management.

The program allows users to create, update and access student records without manually editing CSV files.


## ✨ Features

* Create a new student database
* Add any number of subjects
* Store:

  * Student Name
  * Age
  * Symbol Number
  * Marks in each subject
* Automatically calculate:

  * Total Marks
  * GPA
* Save everything into a CSV file
* View an existing student's record
* Update any existing data
* Delete entire columns if needed
* Display average marks of each subject
* Display average GPA of the class

## 🛠 Technologies Used

* CSV File Handling

## ▶️ How to Run

1. Install Python.
2. Install Pandas if you don't already have it.
pip install pandas
3. Run the program.
python student_record_manager.py
4. Follow the instructions shown in the terminal.
